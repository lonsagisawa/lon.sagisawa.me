## Usage

```shell
# .env
CONTENTFUL_SPACE_ID={Contentful space ID}
CONTENTFUL_DELIVERY_TOKEN={Contentful delivery token}
CONTENTFUL_PREVIEW_TOKEN={Contentful preview token}
```

```shell
pnpm i
# Development
pnpm run dev
# Build
pnpm run build
# Serve
pnpm run preview
```
